# My Claude Project

Bu proje Claude Code ile GitHub entegrasyonunu göstermek için oluşturulmuştur.

## Özellikler

- Claude Code ile otomatik code review
- GitHub Actions ile CI/CD pipeline
- Issue ve Pull Request otomasyonu
- Vercel ile otomatik deployment
- Her commit'te canlıya otomatik yayınlanma

## Kurulum

1. Repository'yi clone'layın
2. GitHub'da Claude App'i kurun
3. ANTHROPIC_API_KEY secret'ını ekleyin

## Kullanım

Issue veya PR yorumlarında `@claude` yazarak Claude'a sorular sorabilirsiniz:

```
@claude bu kodu review eder misin?
@claude bu bug'ı nasıl çözebilirim?
```

## Claude Code ile Çalışma

Yerel geliştirme için:

```bash
claude
```

komutuyla Claude Code'u başlatın ve kodunuz üzerinde çalışmaya başlayın.
